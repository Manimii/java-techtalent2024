Métodos string

Ejercicio 1

Ejercicio 1: Contar caracteres y extraer partes de una cadena
        Dada la cadena "JavaScript es increíble", utiliza los métodos length, slice y substr para:
        - Mostrar la longitud de la cadena.
        - Extraer y mostrar la palabra "increíble" usando slice.
        - Extraer y mostrar la palabra "JavaScript" usando substr

Ejercicio 2

Ejercicio 2: Modificar y formatear cadenas
        Dada la cadena " Bienvenido a la programación ", utiliza los métodos trim, toUpperCase, replace y concat para:
        - Eliminar los espacios en blanco al principio y al final.
        - Convertir la cadena a mayúsculas.
        - Reemplazar la palabra "programación" por "JavaScript".
        - Concatenar la cadena resultante con " en 2024!".

Ejercicio 3

Ejercicio 3: Trabajar con caracteres y subcadenas
        Dada la cadena "Aprender JavaScript es divertido", utiliza los métodos charAt, charCodeAt, split y padEnd para:
        - Mostrar el carácter en el índice 8.
        - Mostrar el código Unicode del carácter en el índice 8.
        - Dividir la cadena en un array de palabras y mostrar el array resultante.
        - Rellenar la cadena original al final con signos de exclamación ("!") hasta alcanzar una longitud de 40 caracteres.
